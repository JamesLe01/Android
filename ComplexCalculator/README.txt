Complex Calculator

“Complex Calculator” Android App - Features including Probability (Calculate Mean, Variance, PDF, CDF, PMF of various commonly used probability distribution); Calculate special functions like Beta, Error, Factorial and Gamma, Combinatorics (Combination and Permutation).
The UI is intuitive, you will know how to navigate through the app as you open it!

The entire folder /ComplexCalculator/ is an Android Studio Project, so you can open this folder using Android Studio and build the APK to install to any Android device.
